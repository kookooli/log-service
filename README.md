## Description

# log-service
A NestJs app that searches treats log files/ Node v19.2.0/ Fastify as http provider

## Installation

```bash
$ npm install
```

```bash
$ npm install -g @nestjs/cli
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
